# Bite Mate – MERN Food Ordering App

Bite Mate is a full-stack MERN application where users can explore food products, like and comment on items, and make payments using Stripe.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS
- Stripe

## Features
- User Authentication
- Product CRUD
- Image Upload
- Like & Comment System
- Stripe Payment Integration
